программа для преобразования двух числовых значений в формат: "инв.№число1, зав.№число2;"

при использовании кнопки "преобразовать из буфера обмена и скопировать" программа получает скопированный текст, преобразовывает и копирует его.
при использовании кнопки "вставить из буфера обмена" текст вставляется, но не копируется.
кнопка "скопировать всё" копирует весь выходной текст, а кнопка "скопировать выделенное" копирует выделенный текст в блоке с выходным текстом, если он там есть.

при помощи контекстного меню наверху можно настроить программу:
во вкладке "Программа" находится настройка "поверх всех окон" которая устанавливает программу поверх всех окон; в той же вкладке находится кнопка выхода из программы.
во вкладке "вставка" указаны сочетания клавиш, которые работают когда программа активна.
с помощью вкладки "добавить пробел" можно указать что в выходном тексте будет указан пробел с нужной стороны.

осторожно! все настройки сохраняются при выходе из программы, и восстанавливаются при запуске.

в случае, если не запускается преобразователь, можно воспользоваться установщиком. однако он установит программу на компьютер, и останутся следы использования.
